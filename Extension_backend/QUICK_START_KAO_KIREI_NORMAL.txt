📝 HOW USERS ADD KAO KIREI SITES:

  1. Open Extension
  2. Go to "Sites" tab
  3. Click "+ Add Site"
  4. Enter URL:
     • https://www.kao-kirei.com/ja/expire-item/khg/?tw=khg
     OR
     • https://www.kao-kirei.com/ja/expire-item/kbb/?tw=kbb
  5. Set check interval (e.g., 24 hours)
  6. Click "Add"
  
  ✅ Done! User will receive notifications for this site.
